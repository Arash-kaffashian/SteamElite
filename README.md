# Introduction
Backend of e-commerce site on my own that customers can register on it and order games they want to have.

# Goal
This project main goal is to provide a way out for following reasons:
- Iranian Steam clients that can't purchase in dollar
- high risk in trading with unknown trades
- Iranian high price marketplaces
